# algo-genetic-fr

Essai de création d'un algorithme génétique en Python.  
## Problème posé :
Un sac a une taille définie, on souhaite y faire rentrer des objets de diverses tailles, de façon à ce que la totalité du sac soit remplie. On ne peut dépasser la taille du sac.  

-------------------------------------------------------------------------------------------------------------------

## Essai de résolution avec un algorithme génétique :  
* A partir d'une population composée de chromosomes qui représentent les objets à mettre dans le sac.  
* On sélectionne les individus qui s'approchent le plus de la valeur du sac, avec une précision donnée.  
* On effectue des croisements sur ces individus.  
* On effectue des mutations aléatoires sur ces individus.  
* Dès qu'un individu valide la cible, ou que les individus ont tous été détruits par la sélection, on arrête le programme.
